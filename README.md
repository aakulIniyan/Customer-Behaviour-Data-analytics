# Customer Behaviour Analysis

## Project Description
This project explores customer shopping behavior using an unclean dataset of retail transactions. The analysis aims to uncover patterns in customer purchases, segment shoppers, and identify trends that can help improve sales strategies and marketing decisions.

## Tools Used
- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- Power BI (for visualization)

## Project Structure
- `Raw unclean data/customer_shopping_behavior.csv` - original dataset used for analysis
- `python code for coding/first_assignment (2).ipynb` - Jupyter Notebook containing data cleaning, analysis, and visualizations
- `Visualization/firstassignment.pbix` - Power BI file for interactive dashboards
- `Step by Step process/first project (2).docx` - project documentation and workflow notes

## Abstract
This study examines customer purchase data to reveal key insights about shopping habits, product preferences, and seasonal demand. The project includes data cleaning, exploratory data analysis, and visual reporting to highlight actionable trends. The results can support business decisions such as targeted promotions, inventory planning, and customer segmentation strategies.

## How to Use
1. Open `python code for coding/first_assignment (2).ipynb` in Jupyter Notebook.
2. Run the notebook cells to reproduce the data cleaning and analysis.
3. Open `Visualization/firstassignment.pbix` in Power BI to view the interactive dashboards.

## Notes
- Ensure Python dependencies are installed before running the notebook.
- The dataset may require preprocessing steps included in the notebook.
